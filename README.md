## 安装

首先安装yeoman

```bash
npm install -g yo
```

然后安装本生成器

```bash
npm install -g generator-mis
```

## 使用

### 生成项目

在模块根目录执行：

```bash
yo mis
```

然后，请跟随贴心制作好的向导一步一步来。

### 生成一个view

在模块根目录执行：

```bash
yo mis:view <view名>
```
再也不用手动去创建勒！
